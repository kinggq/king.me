### 练习题

＊＜丁宁形＞改成＜普通形＞

1、買いません　　　　　　　　　　→　買わない

2、ひまです　　　　　　　　　　　→　ひまだ

3、元気じゃありませんでした　　　→　元気じゃなかった

4、寒くなかったです　　　　　　　→　寒くなかった

5、寝ませんでした　　　　　　　　→　寝なかった

### 普通形　＋　んです的用法

意思：通常用于解释、说明、询问或提供理由，使语气更加自然和亲切。这种结构的使用可用于口语和书面语中，用于解释或强调事实、原因、感情等。

1、为什么不吃蔬菜呢？「问理由」

  - 吃 - 食べ**ません**　→ 　食べ**ない**

  - どうして野菜を食べないんですか

2、是因为不喜欢吃蔬菜「说明理由」

  - 吃 - 嫌い（な形容词）→ 嫌いだ（普通形）　→ 　嫌いな

  - 野菜が嫌いなんです

A: 为什么迟到呢

  - 迟到 - 遅（おく）れました　→ 遅れた

  - どうして遅れたんですか

 B: 是因为公车没有来

  - 来 - 来（き）ませんでした　→ 来（こ）なかった

  - バスが来なかったんです

#### 练习题

A: どうしたんですか（你怎么了）

B: ちょっと頭（あたま）が痛（いた）いんです（头有点痛）

 ——

A: どこか行くんですか　（您要去哪里吗？）

B:　今（いま）から日本へ旅行（りょこう）に行くんです（我现在要去日本旅行）
 
### 普通形　＋　んですが的用法

意思：通常用于解释或者提供理由，并在询问或陈述时使用。这个结构可以使句子更加礼貌和自然。

1、我下个月要去日本，能不能告诉我好的饭店

  - 去 - 行き**ます**　→ 　**行く**

  - 来月（らいげつ）日本へ行くんですが、いいホテルを紹介（しょうかい）してもらえませんか

2、我不知道怎么买车票，可以告诉我吗

  - 不知道 - **分（わ）かりません**　→ 　**分からない**

  - 切符の買い方（かた）が分からないんですが、教えてもらえませんか

3、有点冷，可以帮我关冷气吗

  - 冷 - **寒いです**　→ 　**寒い**

  - 少（すこ）し寒いんですが、クーラーを消（け）してもらえませんか

4、没听到，可以在讲一次吗

  - 听 - **聞こえませんでした**　→ 　**聞こえなかった**

  - 聞こえなかったんですが、もう一度言（いちどい）ってもらえませんか

5、お金（かね）を入（い）れても切符が出（で）ないんですが、どうしたらいいですか （我投了钱，但车票没出来，怎么办呢？）

6、東京駅へ行きたいんですが、どうやって行ったらいいですか（我想去东京站，怎么去比较好呢？）

### 普通形　＋　と思います的用法

意思：表示说话者的个人观点或认为某事是事实，相当于汉语中的“我认为”或“我觉得”等。这个结构用于表达主观看法、意图或推测。

1、我觉得明天会下雨

  - 下雨 - 降（ふ）り**ます**　→ 　**降る**

  - ~~私は~~明日雨が降ると思（おも）います

2、我觉得他不会来

  - 来 - 来**ません**　→ 　**来ない**

  - ~~私は~~彼は来ないと思います

3、我觉得大板很热闹

  - 热闹 - **にぎやかです**　→ 　**にぎやかだ**

  - 大阪（おおさか）はにぎやかだと思います

4、我觉得孩子已经睡觉

  - 睡觉 - **寝ました**　→ 　**寝た**

  - 子供はもう寝たと思います

5、我觉得那个餐厅不好吃

  - 不好吃 - **おいしくありません**　→ 　**おいしくない**

  - あのレストランはおいしくないと思います

6、我觉得这个餐厅很好吃

  - 好吃 - **おいしいです**　→ 　**おいしい**

  - このレストランはとても美味しいと思います

### 普通形　＋　でしょ的用法

意思：吧？

＊な形容词·名词（〜だ　→ 〜~~だ~~）

1、明天会来吧？

  - 来 - き**ます**　→ 　**来（く）る**

  - 明日来るでしょ

2、大阪人很亲切吧

  - 亲切 - **しんせつ**　→ 　**しんせつ~~だ~~**

  - 大阪人（じん）は親切でしょ

3、台湾的水果很好吃吧

  - 好吃 - **おいしいです**　→ 　**おいしい~~**

  - 台湾の果物（くだもの）はおいしいでしょ

4、工作很辛苦吧

  - 辛苦 - **たいへんです**　→ 　**たいへん~~**

  - 仕事（しごと）は大変でしょ

5、很贵吧

  - 贵 - **高（たか）くありません**　→ 　**高くない~~**

  - 高くないでしょ

6、你可能累了吧，让我们休息一下吧

  - 贵 - **高（たか）くありません**　→ 　**高くない~~**

  - 疲（つか）れたでしょ、少し休みましょう

7、日本的樱花很美吧

  - -

  - 日本の桜はきれいでしょ

### 普通形　＋　かどうか的用法

＊な形容词·名词（〜だ　→ 〜~~だ~~）

意思：表示一种疑问，用于询问是否某事情成立或发生。这个结构通常用于间接引语中。

1、不知道明天他会不会来

  - 来 - き**ます**　→ 　**来（く）る**

  - 明日、彼は来るかどうかわかりません

2、不知道那一家餐厅好不好吃

  - 好吃 - **おいしいです**　→ 　**おいしい**

  - あのレストランはおいしいかどうか分りません

3、我不知道明天会不会下雨

  - 下 - **降ります**　→ 　**降る**

  - 明日雨が降るかどうか分かりません

4、因为没见过他，所以不知道漂不漂亮

  - 漂亮 - **きれいです**　→ 　**きれい**

  - 彼女に会ったことがないので、きれいかどうか分かりません

5、因为还没有联络，所以还不知道到没到

  - 到 - **着（つ）きました**　→ 　**着いた**

  - まだ連絡（れんらく）がないので、着いたかどうか分かりません

6、跟他好几年没有见面，所以不知道元气不元气

  - 元气 - **げんきだ**　→ 　**げんき**

  - 彼と何年（なんねん）も会っていないので元気かどうか分かりません

7、因为有一点感冒，所以不知道明天能不能去

  - 去 - **行けます**　→ 　**行ける**

  - ちょっと風邪（かぜ）をひいているので、明日行けるかどうか分かりません
