### い形容词　＋　くなります的用法

意思：这表示随着时间的推移或某种变化，事物的状态变得更加强烈或明显。请注意，这与「い形容词 + くない」的否定形不同，后者表示否定的事实，而「い形容词 + くなります」表示推测。

1、最近变冷了

  - 冷 - さむ**い**　→ 　**さむくなります**

  - 最近寒くなりました

2、这件衣服变便宜了

  - 便宜 - やす**い**　→ 　**やすくなります**

  - この服は安くなりました

3、最近变得很忙了

  - 很忙 - 忙（いそが）しい**い**　→ 　**忙しくなります**

  - 最近忙しくなりました

4、我的儿子长高了

  - 个子高 - 背（せ）が高（たか）**い**　→ 　**背が高くなります**

  - 私の息子（むすこ）は背が高くなりました

5、因为昨天下雪，屋顶变白色了

  - 白色 - 白（しろ）**い**　→ 　**白くなります**

  - 昨日雪（ゆき）が降ったので、屋根（やね）が白くなりました

6、这双鞋变旧了，所以我要扔掉它们

  - 变旧 - 古（ふる）**い**　→ 　**古くなります**

  - この靴（くつ）は古くなったので捨（す）てます

7、女儿长大了，现在几岁了呢

  - 长大 - 大（おお）き**い**　→ 　**大きくなります**

  - 娘（むすめ）さん大きくなりましたね、今何歳（いまなんさい）ですか

  - 今年（ことし）で五歳です

### な形容词和名词　＋　になります的用法

意思：表示某物变得成为某种状态或性质

1、她变漂亮了

  - 漂亮 - きれ**い**　→ 　**きれいになります**

  - 彼女は綺麗になりました

2、变夏天了

  - 夏天 - 夏（なつ）　→ 　**夏になります**

  - 夏になりました

3、因为大家回家了，所以变安静了

  - 安静 - 静（しず）か**な**　→ 　**静かになります**

  - みんな帰（かえ）ったので静かになりました

4、已经晚上了，差不多回家

  - 晚上 - 夜（よる）　→ 　**夜になります**

  - ~~もう~~夜になりました、そろそろ帰りましよう

5、我将来想当棒球选手

  - 棒球选手 - 野球（やきゅう）の選手（せんしゅ）　→ 　**野球の選手になります**

  - 将来（しょうらい）野球の選手になりたいです

6、由于附近新开了一家便利店，变得更加方便了

  - 方便 - べんり　→ 　**べんりになります**

  - 近（ちか）くにコンビニができたので、便利になりました

7、这个公园在春天时樱花会开放

  - 春天 - 春（はる）　→ 　**春になります**

  - この公園は春になると、桜が咲（さ）ます

### 动词辞书形　＋　ようになります的用法

意思：表示“变得可以”或“变得能够”的意思。这个结构用来表示过去不能做某事，而经过一段时间或经过某种变化之后，现在可以做了。

1、我变得能吃蔬菜了（以前不能吃蔬菜）

  - 吃 - 食べられる（可能形）　→ 　**食べられるようになります**

  - 野菜が食べられるようになりました

2、我变得会讲日语了（以前不会讲日语）

  - 讲 - 話せる（可能形）　→ 　**話せるようになります**

  - 日本語が話せるようになりました

3、婴儿变得会走路了

  - 走路 - 歩ける　→ 　**歩けるようになります**

  - 赤（あか）ちゃんが歩けるようになりました

4、我变得会喝酒了

  - 喝 - 飲める　→ 　**飲めるようになります**

  - お酒が飲めるようになりました

5、终于变得会开车了

  - 开 - 運転できる　→ 　**運転できるようになります**

  - やっと運転できるようになりました

6、变得会用日语写信了

  - 写 - 書ける　→ 　**書けるようになります**

  - 日本語で手紙（てがみ）が書けるようになりました

7、最近，我母亲学会了使用电脑

  - 使用 - 使える　→ 　**使えるようになります**

  - 私の母に最近、パソコンが使えるようになりました

### 动词ない形　＋　くなります的用法

意思：通常表示变得不再做某事、不再具有某种特性。这种结构强调状态的改变。

1、最近变得不吃肉了（以前会吃肉）

  - 吃 - 食べない　→ 　**食べなくなります**

  - 最近肉（にく）を食べなくなりました

2、最近变得不运动了（以前会运动）

  - 运动 - 運動しない　→ 　**運動しなくなります**

  - 最近運動（うんどう）をしなくなりました

3、生病之后变得不喝酒了

  - 喝 - 飲まない　→ 　**飲まなくなります**

  - 病気（びょうき）になってから、お酒を飲まなくなりました

4、最近很少买衣服了

  - 买 - 買わない　→ 　**買わなくなります**

  - 最近、あまり服を買わなくなりました

5、最近很少去KTV了

  - 去 - 行かない　→ 　**行かなくなります**

  - 最近、あまりカラオケへ行かなくなりました

6、回台湾后很少讲日语了

  - 讲 - 話さない　→ 　**話さなくなります**

  - 台湾へ帰ってから、日本語をあまり話さなくなりました

7、因为感冒，所以变得无法去旅行

  - 去 - 行けない　→ 　**行けなくなります**

  - 風邪（かぜ）をひいたので、旅行（りょこう）へ行けなくなりました

### い形容词　＋　くします的用法

意思：表示使某物变得某种状态。这种结构通常用于表达主观感受或意愿

1、请你把声音变大

  - 大 - 大（おお）きい　→ 　**大きくします**

  - 音（おと）を大きくしてください

2、请你弄便宜点

  - 便宜 - やすい　→ 　**やすくします**

  - 安くしてください

3、请你弄辣一点

  - 辣 - 辛（から）い　→ 　**辛くします**

  - 辛くしてください

4、房间有点暗，请你弄亮一点

  - 亮 - 明（あか）るい　→ 　**明るくします**

  - 部屋（へや）が暗（くら）いので、明るくしてください

5、还有点长，请你在弄短一点

  - 短 - 短（みじか）い　→ 　**短くします**

  - まだ長（なが）いので、もう少（すこ）し短くしてください

6、椅子有点低，请再调高一点

  - 高 - 高（たか）い　→ 　**高くします**

  - 椅子（いす）が少し低（ひく）いので、もう少し高くしてください

7、请再加一点糖，让它更甜

  - 甜 - 甘（あま）い　→ 　**甘くします**

  - もう少し砂糖（さとう）を入（い）れて、甘くしてください

### な形容词和名词　＋　にします的用法

意思：表示决定将某物变成某种状态或情况

1、请你把房间弄干净

  - 干净 - きれい　→ 　**きれいにします**

  - 部屋をきれいにしてください

2、我妈妈的梦想是让我当医生

  - 医生 - 医者（いしゃ）　→ 　**医者にします**

  - 私の母の夢（ゆめ）は私を医者にすることです

3、请你安静

  - 安静 - 静（しず）かな　→ 　**静かにします**

  - 静かにしてください

4、我想把这个城市弄热闹一点

  - 热闹 - にぎやかな　→ 　**賑やかにします**

  - 私はこの街（まち）を賑やかにしたいです

5、请你把这个中文翻译成日语

  - 日语 - 日本語　→ 　**日本語にします**

  - この中国語を日本語にしてください

6、我想让我的孩子成为一名棒球选手

  - 选手 - 選手（せんしゅ）　→ 　**選手にします**

  - 私は子供（こども）を野球の（やきゅう）選手にしたいです

7、我一定会让我的女朋友幸福的

  - 幸福 - 幸（しあわ）せ　→ 　**幸せにします**

  - 私は彼女を必（かなら）ず幸せにします

### 动词て形　＋　きました的用法

意思：表示动作的开始，并且动作对说话者产生了某种影响。这种结构用于强调动作的开始，通常伴随着一些变化或影响。

1、开始肚子饿了

  - 饿 - 空（す）きます　→ 　**空いて**

  - お腹（なか）が空いてきました

2、开始下起雨来了

  - 下 - 降ります　→ 　**降って**

  - 雨が降ってきました

3、最近开始变冷了

  - 变冷 - 寒くなります　→ 　**寒くなって**

  - 最近、寒くなってきました

4、开始困了

  - 困 - 眠（ねむ）くなります　→ 　**眠くなって**

  - 眠くなってきました

5、天空慢慢变亮了

  - 亮 - 明（あか）るくなります　→ 　**明るくなって**

  - 空（そら）が明るくなってきました

6、终于身体状况好转了

  - 好转 - よくなります　→ 　**よくなって**

  - やっと体（からだ）の調子（ちょうし）がよくなってきました

7、因为吃得太多，所以开始感到肚子疼痛

  - 好转 - よくなります　→ 　**よくなって**

  - 食べ過（す）ぎたので、お腹（なか）が痛（いた）くなってきました
