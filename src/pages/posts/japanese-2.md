### 动词的种类

| 第一类动词 | 第二类动词 |  | 第三类动词 | 
| -------- | ---- | ---- | --------- |
| i ます形 | e　ます形 | i ます形  |     |
| か**き**ます　| おし**え**ます | お**き**ます | します |
| の**み**ます |　あ**け**ます |　お**り**ます |　きます　|
| あそ**び**ます |　た**べ**ます |　**み**ます | - |
|　と**り**ます　|　**ね**ます | - | - |
|　ま**ち**ます　| - | - | - | - |
|　か**い**ます　| - | - | - | - |
|　はな**し**ます　| - | - | - |

> [!TIP]
> 第一类动词：ます　前面的母音是 **い**　的音
>
> 第二类动词：ます　前面的母音是 **え**　的音
>
> 第三类动词：します　和　きます

### ます形　＋　たい・たくない的用法

意思：想做某些事情

接续：动词 ます 形 + たい/たくない

1、想去日本

- 去 - 行き**ます**　→ 　行き**たい**

- 日本へ行きたいです

2、我不想吃面包

 - 吃 - 食べ**ます**　→ 　食べ**たくない**

 - パンを食べたくないです

3、我不想喝咖啡

 - 喝 - 飲み**ます**　→ 　飲み**たくない**

 - コーヒーを飲みたくないです

4、今天不想工作

 - 工作 - 働（はたら）き**ます**　→ 　働き**たくない**

 - 今日は働きたくないです

5、我想买新衣服

 - 买 - 買い**ます**　→ 　買い**たい**

 - 新（あたら）しい服（ふく）を買いたいです

6、暑假想去海边游泳

 - 游泳 - 泳（およ）ぎ**ます**　→ 　泳ぎ**たい**

 - 夏休（なつやす）みは海（うみ）で泳ぎたいです

7、今天有点累了、所以不想学习。

 - 学习 - 勉強（べんきよう）し**ます**　→ 　勉強し**たくない**

 - 今日は疲（つか）れたので勉強したくないです

 ### ます形　＋　ながら的用法

意思：一边xxx，一边xxx

接续：动词 ます 形 + ながら

1、一边听音乐一边学习

 - 听 - 聞き**ます**　→ 　聞き**ながら**

 - 音楽（おんがく）を聞きながら勉強します

2、一边读报纸一边吃饭

 - 读 - 読（よ）み**ます**　→ 　読み**ながら**

 - 新聞（しんぶん）を読みながらご飯（ごはん）を食べます

3、一边看电视一边抽烟

 - 看 - 見**ます**　→ 　見**ながら**

 - テレビを見ながらタバコを吸います

4、一边唱歌一边洗澡

 - 唱 - 歌（うた）い**ます**　→ 　歌い**ながら**

 - 歌を歌いながらシ（し）ヤワーを浴（あ）びます

5、一边弹吉他一边唱歌

 - 弹 - 弾（ひ）き**ます**　→ 　弾き**ながら**

 - ギターを弾きながら歌を歌います

6、一边跟朋友聊天一边回家

 - 聊 - 話（はな）します　**ます**　→ 　話し**ながら**

 - 友達と話しながら家（うち）に帰（かえ）りました

7、他一边走路一边喝果汁

 - 走 - 歩（ある）き**ます**　→ 　歩き**ながら**

 - 彼は歩きながらジユース（じゆ~す）を飲んでいます

### ます形　＋　やすい・にくい的用法

意思：やすい 或 にくい 构成的形容词，表示某事物做起来容易或困难

1、这支笔很好写

 - 写 - 書き**ます**　→ 　書き**やすい**

 - このペンは書きやすいです

2、这支笔不好写

 - 写 - 書き**ます**　→ 　書き**にくい**

 - このペンは書きにくいです

3、这个药容易吃

 - 吃 - 飲み**ます**　→ 　飲み**やすい**

 - この薬（くすり）は飲み易いです

4、铃木老师的课容易懂

 - 懂 - 分（わ）かり**ます**　→ 　分かり**やすい**

 - 鈴木（すずき）先生の授業（じゆぎよう）は分かり易い

5、这部手机很难用

 - 用 - 使（つか）い**ます**　→ 　使い**にくい**

 - この携帯電話（けいたいでんわ）は使いにくいです

6、这首歌很简单，所以很容易唱

> [!TIP]
> 在这个句子中，「なので」是一个表示原因或理由的连词，通常翻译为“所以”或“因此”。
>
>「この歌は簡単な」表示“这首歌很简单”；
>
>「ので」连接前后两个句子，表示因果关系，翻译为“所以”。

 - 唱 - 歌い**ます**　→ 　歌い**やすい**

 - この歌は簡単（かんたん）なので歌いやすいです

7、这个包包不好拿

 - 拿 - まち**ます**　→ 　まち**にくい**

 - このかばんはまちにくいです

### ます形　＋　かた的用法

意思：这个结构常用于询问、表达、学习某种技能或动作的方法

1、不知道怎么用（使用的方法）

 - 用 - 使い**ます**　→ 　使い**かた**

 - 使い方（かた）が分（わ）かりません

2、请你教我怎么买票

 - 买 - 買い**ます**　→ 　買い**かた**

 - 切符（きっぷ）の買い方を教（おし）えて下（くだ）さい

3、这个料理的做法很简单

 - 做 - 作り**ます**　→ 　作り**かた**

 - この料理を作り方は簡単です

4、请教我东京车站怎么走

 - 走 - 行き**ます**　→ 　行き**かた**

 - 東京駅（とうきようえき）の行き方を教えてください

5、不知道怎么学日文

 - 学 - し**ます**　→ 　し**かた**

 - 日本語の勉強のしかたが分りません

6、对不起、能告诉我如何复制吗

 - 做 - し**ます**　→ 　し**かた**

 - すみませんが、コーピのしかたを教えて下さい

7、老师教的方法很厉害

 - 教 - 教え**ます**　→ 　教え**かた**

 - 先生の教え方はとても上手です

### ます形　＋　始めます的用法

意思：开始做一个事情

1、今天开始写日记

 - 写 - 書き**ます**　→ 　書き**始（はじ）めます**

 - 今日から日記（にっき）を書き始めます

2、樱花是三月开始开花

 - 开 - 咲（さ）き**ます**　→ 　咲き**始めます**

 - 桜（さくら）は三月（さんがつ）から咲き始めます

3、她从四月开始工作了

 - 工作 - 働（はたら）き**ます**　→ 　働き**始めます**

 - 彼女は四月（しがつ）から働き始めました

4、最近开始读日语书了

 - 读 - 読み**ます**　→ 　読み**始めます**

 - 最近（さいきん）、日本語の本を読み始めました

5、刚开始吃午餐的时候，朋友来到了我的家

 - 吃 - 食べ**ます**　→ 　食べ**始めます**

 - お昼（ひる）ごはんを食べはじめたら、友達が家（うち）に来ました

6、他二十五岁开始抽烟了

 - 抽 - 吸い**ます**　→ 　吸い**始めます**

 - 彼は二十五歳（にじゆうごさい）でタバコを吸い始めました

7、最近，我开始考虑将来的事情了

 - 考虑 - 考え（かんがえ）**ます**　→ 　考え**始めます**

 - 最近、将来（しようらい）の事（こと）について考え始めました

### ます形　＋　終わります的用法

 意思：表示某个动作或状态的完成

1、老师、我写完了

 - 写 - 書き**ます**　→ 　書き**終（お）わります**

 - 先生、書き終わりました

2、吃完了吗

 - 吃 - 食べ**ます**　→ 　食べ**終わります**

 - 食べ終わりましたか

3、终于看完这本书了

 - 看 - 読み**ます**　→ 　読み**終わります**

 - やっとこの本を読み終わりました

4、洗完衣服了吗

 - 洗 - 洗（あら）い**ます**　→ 　洗い**終わります**

 - 服を洗い終わりましたか

5、讲完了吗

 - 讲 - 話し**ます**　→ 　話し**終わります**

 - 話し終わりましたか

6、抽完烟走吧

 - 抽 - 吸い**ます**　→ 　吸い**終わります**

 - タバコを吸い終わったら行きましよう

7、请你等我喝完这杯咖啡

 - 喝 - 飲み**ます**　→ 　飲み**終わります**

 - このコーヒを飲み終わるまで待（ま）って下（くだ）さい

### ます形　＋　続けます的用法

意思：持续性的动作或行为

1、等朋友等了一个小时

 - 等 - 待ち**ます**　→ 　待ち**続（つづ）けます**

 - 友達を一時間（いちじかん）も待ち続けました

2、昨天唱歌唱了五个小时

 - 唱 - 歌い**ます**　→ 　歌い**続けます**

 - 昨日（きのう）、五時間（ごじかん）も歌い続けました

3、他从大阪到神户一直走路

 - 走 - 歩（ある）き**ます**　→ 　歩き**続けます**

 - 彼は大阪（おおさか）から神戸まで歩き続けました

4、他一直吃，直到肚子饱了为止

 - 吃 - 食べ**ます**　→ 　食べ**続けます**

 - 彼はお腹（おな）がいっぱいになるまで食べ続けました

> [!TIP]
> 「彼」（かれ）：他
> 
> 「は」：主题标记，表示主语
> 
> 「お腹」（おなか）：肚子
> 
> 「が」：格助词，表示主语
> 
> 「いっぱい」：满，饱
>
> 「になる」：成为
>
> 「まで」：直到

5、这个电脑我会用到坏掉为止

 - 用 - 使い**ます**　→ 　使い**続けます**

 - このパソコンは壊（こわ）れるまで使い続けます

6、昨天一整天都在写报告

 - 写 - 書き**ます**　→ 　書き**続けます**

 - 昨日一日中（いちにちじゆう）レポートを書き続けました

7、我一直道歉，直到女朋友原谅为止

 - 道歉 - 謝（あやま）り**ます**　→ 　謝り**続けます**

 - 彼女が許（ゆる）してくれるまで謝り続けました

### ます形　＋　出します的用法

意思：某个动作或状态的发生

1、突然下起雨来了

 - 下 - 降（ふ）り**ます**　→ 　降り**出（だ）します**

 - ~~急（きゆう）に~~雨（あめ）が降り出しました

2、婴儿突然哭起来了

 - 哭 - 泣（な）き**ます**　→ 　泣き**出します**

 - ~~急（きゆう）に~~赤（あか）ちゃんが泣き出しました

3、老师突然生气起来了

 - 生气 - 怒（おこ）り**ます**　→ 　怒り**出します**

 - ~~急（きゆう）に~~先生が怒り出しました

4、车子突然动起来了

 - 动 - 動（うご）き**ます**　→ 　動き**出します**

 - ~~急（きゆう）に~~車が動き出しました

5、电话突然响起来了

 - 响 - 鳴（な）り**ます**　→ 　鳴り**出します**

 - ~~急（きゆう）に~~電話が鳴り出しました

6、他听到一些事情后突然跑起来了

 - 跑 - 走（はし）り**ます**　→ 　走り**出します**

 - 彼は話（はな）しを聞いた瞬間（しゆんかん）急に走り出しました

7、突然肚子痛起来了

 - 痛 - 痛（いた）く**ます**　→ 　痛く**出します**

 - ~~急（きゆう）に~~お腹が痛くなり出しました

### ます形　＋　に行きます的用法

意思：将要去做某事或前往某地

1、去百货公司买衣服

 - 买 - 買い**ます**　→ 　買い**に行きます**

 - デパート<ruby>へ<rt style="font-size: 12px;">え</rt></ruby>服を買いに行きます

2、去日本看樱花

 - 看 - 見**ます**　→ 　見**に行きます**

 - 日本へ桜を見に行きます

3、我要去居酒屋喝酒

 - 喝 - 飲み**ます**　→ 　飲み**に行きます**

 - 居酒屋（いざかや）へ~~お酒を~~飲みに行きます

4、我要去图书馆借书

 - 借 - 借（か）り**ます**　→ 　借り**に行きます**

 - 図書館（としよかん）へ本を借りに行きます

5、我要去朋友家玩儿

 - 玩 - 遊び**ます**　→ 　遊び**に行きます**

 - 友達の家（うち）へ遊びに行きます

6、去外面抽烟

 - 抽 - 吸い**ます**　→ 　吸い**に行きます**

 - 外（そと）へタバコを吸いに行きます

7、明天去学校见朋友

 - 见 - 会（あ）い**ます**　→ 　会い**に行きます**

 - 明日、学校（がっこう）へ友達に会いに行きます

 - ~~一緒に（いっしよ）~~　コーヒーを飲みませんか

 - 玩 - 遊び**ます**　→ 　遊び**ませんか**

### ます形　＋　ましようか的用法

意思：表示邀请或提议，用于表达对某种行为的建议或邀请。这种结构通常用于礼貌地提出建议或询问对方的意愿。

1、要不要帮你开窗户

 - 开 - 開（あ）け**ます**　→ 　開け**ましようか**

 - 窓（まど）開けましようか

2、要不要帮你拿行李

 - 拿 - 待（ま）ち**ます**　→ 　待ち**ましようか**

 - 荷物（にもつ）を待ちましようか

3、要不要帮忙

 - 帮忙 - 手伝（てつだ）い**ます**　→ 手伝い**ましようか**

 - 手伝いましようか

4、要不要帮你丢垃圾

 - 丢 - 捨（す）て**ます**　→ 捨て**ましようか**

 - ごみを捨てましようか

5、要不要帮你拍照

 - 拍 - 撮り**ます**　→ 撮り**ましようか**

 - 写真（しやしん）を撮りましようか

6、要不要帮你把资料复印一下

 - 做 - し**ます**　→ し**ましようか**

 - その資料（しりよう）をコピーしましようか

7、要不要帮你泡咖啡

 - 泡 - いれ**ます**　→ いれ**ましようか**

 - コーヒーをいれましようか
