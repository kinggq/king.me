### 动词的种类

| 第一类动词 | 第二类动词 |  | 第三类动词 | 
| -------- | ---- | ---- | --------- |
| i ます形 | e　ます形 | i ます形  |     |
| か**き**ます　| おし**え**ます | お**き**ます | します |
| の**み**ます |　あ**け**ます |　お**り**ます |　きます　|
| あそ**び**ます |　た**べ**ます |　**み**ます | - |
|　と**り**ます　|　**ね**ます | - | - |
|　ま**ち**ます　| - | - | - | - |
|　か**い**ます　| - | - | - | - |
|　はな**し**ます　| - | - | - |

> [!TIP]
> 第一类动词：ます　前面的母音是 **い**　的音
>
> 第二类动词：ます　前面的母音是 **え**　的音
>
> 第三类动词：します　和　きます

### ます形　＋　たい・たくない的用法

意思：想做某些事情

接续：动词 ます 形 + たい/たくない

1、想去日本

- 去 - 行き**ます**　→ 　行き**たい**

- 日本へ行きたいです

2、我不想吃面包

 - 吃 - 食べ**ます**　→ 　食べ**たくない**

 - パンを食べたくないです

3、我不想喝咖啡

 - 喝 - 飲み**ます**　→ 　飲み**たくない**

 - コーヒーを飲みたくないです

4、今天不想工作

 - 工作 - 働（はたら）き**ます**　→ 　働き**たくない**

 - 今日は働きたくないです

5、我想买新衣服

 - 买 - 買い**ます**　→ 　買い**たい**

 - 新（あたら）しい服（ふく）を買いたいです

6、暑假想去海边游泳

 - 游泳 - 泳（およ）ぎ**ます**　→ 　泳ぎ**たい**

 - 夏休（なつやす）みは海（うみ）で泳ぎたいです

7、今天有点累了、所以不想学习。

 - 学习 - 勉強（べんきよう）し**ます**　→ 　勉強し**たくない**

 - 今日は疲（つか）れたので勉強したくないです

 ### ます形　＋　ながら的用法

意思：一边xxx，一边xxx

接续：动词 ます 形 + ながら

1、一边听音乐一边学习

 - 听 - 聞き**ます**　→ 　聞き**ながら**

 - 音楽（おんがく）を聞きながら勉強します

2、一边读报纸一边吃饭

 - 读 - 読（よ）み**ます**　→ 　読み**ながら**

 - 新聞（しんぶん）を読みながらご飯（ごはん）を食べます

3、一边看电视一边抽烟

 - 看 - 見**ます**　→ 　見**ながら**

 - テレビを見ながらタバコを吸います

4、一边唱歌一边洗澡

 - 唱 - 歌（うた）い**ます**　→ 　歌い**ながら**

 - 歌を歌いながらシ（し）ヤワーを浴（あ）びます

5、一边弹吉他一边唱歌

 - 弹 - 弾（ひ）き**ます**　→ 　弾き**ながら**

 - ギターを弾きながら歌を歌います

6、一边跟朋友聊天一边回家

 - 聊 - 話（はな）します　**ます**　→ 　話し**ながら**

 - 友達と話しながら家（うち）に帰（かえ）りました

7、他一边走路一边喝果汁

 - 走 - 歩（ある）き**ます**　→ 　歩き**ながら**

 - 彼は歩きながらジユース（じゆ~す）を飲んでいます

### ます形　＋　やすい・にくい的用法

意思：やすい 或 にくい 构成的形容词，表示某事物做起来容易或困难

1、这支笔很好写

 - 写 - 書き**ます**　→ 　書き**やすい**

 - このペンは書きやすいです

2、这支笔不好写

 - 写 - 書き**ます**　→ 　書き**にくい**

 - このペンは書きにくいです

3、这个药容易吃

 - 吃 - 飲み**ます**　→ 　飲み**やすい**

 - この薬（くすり）は飲み易いです

4、铃木老师的课容易懂

 - 懂 - 分（わ）かり**ます**　→ 　分かり**やすい**

 - 鈴木（すずき）先生の授業（じゆぎよう）は分かり易い

5、这部手机很难用

 - 用 - 使（つか）い**ます**　→ 　使い**にくい**

 - この携帯電話（けいたいでんわ）は使いにくいです

6、这首歌很简单，所以很容易唱

> [!TIP]
> 在这个句子中，「なので」是一个表示原因或理由的连词，通常翻译为“所以”或“因此”。
>
>「この歌は簡単な」表示“这首歌很简单”；
>
>「ので」连接前后两个句子，表示因果关系，翻译为“所以”。

 - 唱 - 歌い**ます**　→ 　歌い**やすい**

 - この歌は簡単（かんたん）なので歌いやすいです

7、这个包包不好拿

 - 拿 - まち**ます**　→ 　まち**にくい**

 - このかばんはまちにくいです

### ます形　＋　かた的用法

意思：这个结构常用于询问、表达、学习某种技能或动作的方法

1、不知道怎么用（使用的方法）

 - 用 - 使い**ます**　→ 　使い**かた**

 - 使い方（かた）が分（わ）かりません

2、请你教我怎么买票

 - 买 - 買い**ます**　→ 　買い**かた**

 - 切符（きっぷ）の買い方を教（おし）えて下（くだ）さい

3、这个料理的做法很简单

 - 做 - 作り**ます**　→ 　作り**かた**

 - この料理を作り方は簡単です

4、请教我东京车站怎么走

 - 走 - 行き**ます**　→ 　行き**かた**

 - 東京駅（とうきようえき）の行き方を教えてください

5、不知道怎么学日文

 - 学 - し**ます**　→ 　し**かた**

 - 日本語の勉強のしかたが分りません

6、对不起、能告诉我如何复制吗

 - 做 - し**ます**　→ 　し**かた**

 - すみませんが、コーピのしかたを教えて下さい

7、老师教的方法很厉害

 - 教 - 教え**ます**　→ 　教え**かた**

 - 先生の教え方はとても上手です

### ます形　＋　始めます的用法

意思：开始做一个事情

1、今天开始写日记

 - 写 - 書き**ます**　→ 　書き**始（はじ）めます**

 - 今日から日記（にっき）を書き始めます

2、樱花是三月开始开花

 - 开 - 咲（さ）き**ます**　→ 　咲き**始めます**

 - 桜（さくら）は三月（さんがつ）から咲き始めます

3、她从四月开始工作了

 - 工作 - 働（はたら）き**ます**　→ 　働き**始めます**

 - 彼女は四月（しがつ）から働き始めました

4、最近开始读日语书了

 - 读 - 読み**ます**　→ 　読み**始めます**

 - 最近（さいきん）、日本語の本を読み始めました

5、刚开始吃午餐的时候，朋友来到了我的家

 - 吃 - 食べ**ます**　→ 　食べ**始めます**

 - お昼（ひる）ごはんを食べはじめたら、友達が家（うち）に来ました

6、他二十五岁开始抽烟了

 - 抽 - 吸い**ます**　→ 　吸い**始めます**

 - 彼は二十五歳（にじゆうごさい）でタバコを吸い始めました

7、最近，我开始考虑将来的事情了

 - 考虑 - 考え（かんがえ）**ます**　→ 　考え**始めます**

 - 最近、将来（しようらい）の事（こと）について考え始めました

### ます形　＋　終わります的用法

 意思：表示某个动作或状态的完成

1、老师、我写完了

 - 写 - 書き**ます**　→ 　書き**終（お）わります**

 - 先生、書き終わりました

2、吃完了吗

 - 吃 - 食べ**ます**　→ 　食べ**終わります**

 - 食べ終わりましたか

3、终于看完这本书了

 - 看 - 読み**ます**　→ 　読み**終わります**

 - やっとこの本を読み終わりました

4、洗完衣服了吗

 - 洗 - 洗（あら）い**ます**　→ 　洗い**終わります**

 - 服を洗い終わりましたか

5、讲完了吗

 - 讲 - 話し**ます**　→ 　話し**終わります**

 - 話し終わりましたか

6、抽完烟走吧

 - 抽 - 吸い**ます**　→ 　吸い**終わります**

 - タバコを吸い終わったら行きましよう

7、请你等我喝完这杯咖啡

 - 喝 - 飲み**ます**　→ 　飲み**終わります**

 - このコーヒを飲み終わるまで待（ま）って下（くだ）さい

### ます形　＋　続けます的用法

意思：持续性的动作或行为

1、等朋友等了一个小时

 - 等 - 待ち**ます**　→ 　待ち**続（つづ）けます**

 - 友達を一時間（いちじかん）も待ち続けました
